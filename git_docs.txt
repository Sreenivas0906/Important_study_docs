	How to go back to previous version of code
	git reset --hard f1cd7c3(commitID)
	
	How to revert back recently committed from remote server??
	git revert <recent commitID>
	
	How to revert back previously commited in remote server??
	Its only possible recently committed once. but not possible to revert back previous commits.

	git show --pretty="" --name-only <commitID>  --> it will show only committed file names
