1) What is VS CODE ??
    ** VS CODE (Visual Studio Code) is an open source code editor primarily used to correct and replace cloud and web applications coding errors .
    ** VS Code is developed my Microsoft and supported by the mac OS , Linux and windows operating systems.

programming language that use VS code :
